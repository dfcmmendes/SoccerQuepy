Projecto final Web Semantica

IDE utilizado: 
-PyCharm Community

Dependencias:
-Flask
-Xampp

Como correr o projecto:
-Instalar IDE
-Instalar dependencias
	-Flask: pip install Flask
	-Xampp: instalador do site
-Abrir projecto pelo IDE
-Executar o main.py (SoccerQuepy/examples/dbpedia/main.py) através do IDE
-na consola deverá aparecer
	"Running on http://127.0.0.1:5000/ (Press CTRL+C to quit)" - ou semelhante
-entrar no URL e usar a interface para escrever as queries
